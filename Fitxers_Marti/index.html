Les llistes en CSS són una de les propietats que utilitzem frequentment, per tant és necessari saber utilitzar els estils d'aquests.

  • Valor predeterminat: disc outside none.
  • Herencia: Si.
  • Animable: No. 
            .square {
                list-style: square;
            }
            
            .inside {
                list-style: inside;
            }

Per estructurar bé el css de list-style s'ha de seguir un ordre de estructuració concret.

    /* tipus | imatge | posició */

    list-style: lower-roman url("../img/shape.png") outside;

Hi han varis valors que donen més joc a l'hora de decorar les teves llistes.

  • list-style-type:


  • list-style-image:

    • La propietat estableix una imatge que s'utilitzarà com a marcador d'elements de la llista.


  • list-style-position:


    • Per lo general, tot el que es pot fer en les variacions de type, image i position es pot fer en el mateix list-style
